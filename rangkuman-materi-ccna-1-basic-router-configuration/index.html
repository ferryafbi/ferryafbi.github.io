Rangkuman Materi CCNA-1: Basic Router Configuration

Ferry Ananda Febian

Jan 4, 2023 • 3 min read

Basic Router Configuration adalah proses mengatur dan mengkonfigurasi router agar dapat bekerja sesuai dengan kebutuhan jaringan.

Konfigurasi Pengaturan Awal Router Cisco IOS

Berikut ini adalah langkah-langkah untuk mengkonfigurasi pengaturan awal pada router Cisco IOS:

  1. Masuk ke mode konfigurasi dengan mengetik “enable” dan menekan tombol Enter. Kemudian masukkan password yang sesuai jika diminta.
  2. Masuk ke mode konfigurasi global dengan mengetik “configure terminal” dan menekan tombol Enter.
  3. Konfigurasi hostname router dengan mengetik “hostname [nama host]”. Ganti “[nama host]” dengan nama host yang diinginkan.
  4. Konfigurasi password enable dengan mengetik “enable password [password]”. Ganti “[password]” dengan password yang diinginkan.
  5. Konfigurasi password line console dengan mengetik “line console 0” dan menekan tombol Enter, kemudian masukkan “password [password]” dan “login”. Ganti “[password]” dengan password yang diinginkan.
  6. Konfigurasi interface jaringan dengan mengetik “interface [nama interface]”. Ganti “[nama interface]” dengan nama interface yang sesuai (misalnya “Ethernet 0/0”).
  7. Konfigurasi alamat IP pada interface tersebut dengan mengetik “ip address [alamat IP] [subnet mask]”. Ganti “[alamat IP]” dengan alamat IP yang diinginkan dan “[subnet mask]” dengan subnet mask yang sesuai.

misal :

ip address 192.168.10.10 255.255.255.0
  1. Kembali ke mode konfigurasi global dengan mengetik “end” dan menekan tombol Enter.
  2. Simpan konfigurasi yang telah dibuat dengan mengetik “copy running-config startup-config” dan menekan tombol Enter.

Konfigurasi Dua Interface Aktif pada Router Cisco IOS

Untuk mengkonfigurasi dua interface aktif pada router Cisco IOS, ikuti langkah-langkah berikut:

  1. Masuk ke modus konfigurasi dengan perintah “configure terminal”.
  2. Masuk ke konfigurasi interface dengan perintah “interface nama_interface”, di mana “nama_interface” adalah nama interface yang ingin Anda konfigurasi. Misalnya, jika Anda ingin mengkonfigurasi interface Ethernet 0/0, perintahnya adalah “interface Ethernet 0/0”.
  3. Aktifkan interface dengan perintah “no shutdown”.
  4. Atur alamat IP untuk interface dengan perintah “ip address alamat_ip netmask subnet_mask”. Misalnya, jika Anda ingin memberikan alamat IP “192.168.1.1” dengan subnet mask “255.255.255.0” pada interface, perintahnya adalah “ip address 192.168.1.1 255.255.255.0”.
  5. Ulangi langkah 2 hingga 4 untuk interface yang kedua.
  6. Kembali ke modus konfigurasi teratas dengan perintah “end” atau “exit”.
  7. Simpan konfigurasi dengan perintah “wr” atau “write memory”.

Contoh lengkap:

Router(config)# interface Ethernet 0/0
Router(config-if)# no shutdown
Router(config-if)# ip address 192.168.1.1 255.255.255.0
Router(config-if)# exit
Router(config)# interface Ethernet 0/1
Router(config-if)# no shutdown
Router(config-if)# ip address 192.168.2.1 255.255.255.0
Router(config)# end
Router# wr

Konfigurasi Perangkat Menggunakan Gateway Default

Untuk mengkonfigurasi perangkat agar menggunakan gateway default, Anda bisa mengikuti langkah-langkah berikut:

  1. Masuk ke dalam perangkat yang akan dikonfigurasi.
  2. Buka pengaturan jaringan atau koneksi internet.
  3. Cari pengaturan alamat IP dan subnet mask.
  4. Isikan alamat IP dan subnet mask yang sesuai dengan jaringan yang akan digunakan.
  5. Pada bagian “gateway”, isikan alamat IP dari gateway default yang ingin digunakan.
  6. Simpan perubahan dengan menekan tombol “apply” atau “ok”.

Contoh:

  1. Pada perangkat Windows, buka “Control Panel” dan masuk ke “Network and Internet”.
  2. Pilih “Network and Sharing Center” dan klik pada “Change adapter settings”.
  3. Klik kanan pada koneksi yang ingin Anda konfigurasi, lalu pilih “Properties”.
  4. Pilih “Internet Protocol Version 4 (TCP/IPv4)” dan klik “Properties”.
  5. Pada tab “General”, pilih “Use the following IP address” dan isikan alamat IP dan subnet mask yang sesuai dengan jaringan yang akan digunakan.
  6. Pada bagian “Default gateway”, isikan alamat IP dari gateway default yang ingin digunakan.
  7. Klik “OK” untuk menyimpan perubahan.

Sebagai catatan, pastikan bahwa alamat IP yang Anda masukkan pada perangkat tidak sama dengan alamat IP yang digunakan oleh perangkat lain di jaringan yang sama. Juga pastikan bahwa subnet mask yang Anda masukkan sesuai dengan subnet mask yang digunakan di jaringan tersebut.
