Cara Menampilkan File Tersembunyi di Linux

Ferry Ananda Febian

Nov 22, 2022 • 2 min read

Linux menyediakan banyak fitur yang dapat membantu Anda saat berinteraksi dengan file dan direktori di terminal Linux. Salah satunya yaitu file hidden atau file tersembunyi. Anda dapat menampilkan file tersembunyi dan membuat daftar file yang akan disembunyikan.

Artikel ini akan membahas bagaimana cara menampilkan file tersembunyi dan membuat daftar file yang akan disembunyikan di Linux.

Cara Menampilkan File Tersembunyi di Linux

Tampilkan File Tersembunyi dengan ls

ls adalah singkatan list, dengan ls dan opsi -a (all), Anda dapat menampilkan semua file tersembunyi pada sebuah direktori.

ls -a /path/to/folder

Contoh:

ls -a /home/user/Downloads

Pada perintah di atas, ls akan mencantumkan semua file (termasuk yang tersembunyi) pada path atau direktori yang ditentukan. Jika Anda tidak menentukan path folder, perintah ls akan mencantumkan file yang ada di direktori saat perintah tersebut dieksekusi.

Tampilkan Hanya File yang Tersembunyi

Jika Anda hanya ingin melihat file tersembunyi, gunakan ekspresi reguler untuk memfilter daftar file yang ditampilkan.

ls -dl .[^.]* /path/to/folder

Berikut adalah detail perintah diatas:

Saya menggunakan regex .[^.]* untuk menentukan file & direktori yang dimulai dengan titik (.) di namanya. Ini karena, di Linux, semua file & folder tersembunyi dimulai dengan titik di awal file. Dengan kata lain, Saya meminta perintah ls untuk membuat daftar file & direktori yang namanya dimulai dengan titik.

Anda juga dapat menggunakan salah satu dari perintah berikut untuk mendapatkan hasil yang sama

ls -d .* /path/to/folder

Atau

ls -ld ~/.[^.]* /path/to/folder

Atau

ls -l ~/.??* /path/to/folder

Atau

ls -ld ~/.??* /path/to/folder

Tampilkan File Tersembunyi dengan Find

Anda juga dapat menampilkan file tersembunyi dengan menambahkan -name option

sudo find /directory/to/folder -name ".*" 2> /dev/null

Berikut adalah detail perintah diatas:

Saya menggunakan opsi -name untuk menentukan bahwa Saya ingin menemukan semua file yang dimulai dengan tanda “.”, yaitu file tersembunyi. SAya juga mengarahkan output dari perintah ke /dev/null untuk menghindari output pesan kesalahan karena izin dan akses pengguna yang tidak memadai.

Berikut adalah perintah untuk menemukan semua file tersembunyi di sistem Anda

sudo find / -name ".*" 2> /dev/null

Bisa juga digunakan untuk menemukan semua file tersembunyi di direktori kerja Anda saat ini.

sudo find . -name ".*" 2> /dev/null

Tampilkan File Tersembunyi Hanya dengan Find

Anda dapat menampilkan file tersembunyi hanya dengan menggunakan perintah Find, tambahkan sintaks -type option

sudo find /path/to/folder -name ".*" -type d 2> /dev/null

Tampilkan File Tersembunyi dengan Perintah Dir

Anda dapat menampilkan file tersembunyi dengan perintah Dir, dir mendukung opsi -a dan -A yang termasuk implied dari file dan direktori masing-masing. 

sudo dir -a /path/to/folder
sudo dir -A /path/to/folder

Sama seperti perintah ls, Anda dapat menampilkan hanya file tersembunyi dengan ekspresi reguler :

sudo dir -dl .[^.]* /path/to/folder
