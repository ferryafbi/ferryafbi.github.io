Tipe Data Boolean di PHP

Ferry Ananda Febian

Nov 17, 2022 • 2 min read

Ini adalah tipe data yang paling sederhana, bool hanya bernilai true atau false.

Sintaks

Untuk menentukan menentukan nilai bool, gunakan konstanta true atau false. Keduanya tidak peka huruf besar-kecil.

<?php
$foo = True; // assign the value TRUE to $foo

echo gettype ($foo)

?>

Biasanya, hasil dari operator yang mengembalikan nilai bool akan diteruskan ke struktur kontrol.

<?php

$action = 'show_versi';

$show_pembatas = '<hr>';

// == adalah operator uji

// persamaan dan pengembalian boolean

if ($action == "show_versi") {

    echo "Versi aplikasi adalah 1.2.3 \n";

}

// this is not necessary...

if ($show_pembatas == TRUE) {

    echo "<hr>\n";

}

// ...because this can be used with exactly the same meaning:

echo $show_pembatas."\n"

?>

Konversi ke boolean

Untuk mengkonversi nilai menjadi bool secara eksplisit, gunakan (bool) atau (boolean). Namun, dalam banyak kasus, pemeran tidak diperlukan, karena nilai akan dikonversi secara otomatis jika operator, fungsi, atau struktur kontrol memerlukan argumen bool.

Saat mengkonversi data ke bool, nilai berikut ini dianggap salah:

  • Nilai boolean itu sendiri False
  • Nilai integer 0 (nol)
  • Nilai floats 0.0 dan -0.0 (nol)
  • String kosong, dan string “0”
  • Sebuah array dengan elemen nol
  • Tipe spesial NULL (termasuk variable yang tidak diset)
  • Object sederhana XML dibuat dari atribut elemen yang kosong, yaitu elemen yang tidak memiliki turunan atau atribut.
  • Setiap nilai dianggap true (termasuk resources dan NAN).

Peringatan

-1 dianggap benar, seperti angka bukan nol (baik negatif maupun positif)!

<?php
var_dump((bool) "");        // bool(false)
var_dump((bool) "0");       // bool(false)
var_dump((bool) 1);         // bool(true)
var_dump((bool) -2);        // bool(true)
var_dump((bool) "foo");     // bool(true)
var_dump((bool) 2.3e5);     // bool(true)
var_dump((bool) array(12)); // bool(true)
var_dump((bool) array());   // bool(false)
var_dump((bool) "false");   // bool(true)
var_dump((bool) -1);        // bool(true)
?>
