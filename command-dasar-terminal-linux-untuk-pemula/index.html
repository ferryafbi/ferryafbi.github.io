Command Dasar Terminal Linux untuk Pemula

Ferry Ananda Febian

Aug 13, 2022 • 2 min read

Command Linux

Linux Shell atau “Terminal” pada dasarnya adalah program yang menerima perintah dari pengguna dan langsung diproses oleh operating system atau sistem operasi , lalu akan menghasilkan output dari proses tersebut. Distro linux hadir dalam versi GUI graphical user interface atau (antarmuka pengguna grafis) seperti Ubuntu, Red Hat, Mint, Debian, Fedora, tetapi pada dasarnya semua distro linux memiliki CLI atau command line interface yang berarti (antarmuka baris perintah).

pwd

atau Print Working Directory ini digunakan untuk mengetahui posisi kita ada dimana.

pwd

ls

atau listing digunakan untuk menampilkan semua daftar file/folder yang ada di sebuah direktori.

ls

cd

digunakan untuk masuk ke sebuah direktori. dan jika ingin keluar dari direktori gunakan cd spasi titik titik.

cd [nama-direktori]
cd Desktop

mkdir, rmdir

mkdir digunakan untuk membuat sebuah folder atau direktori kosong dan rmdir digunakan untuk menghapus folder atau direktori yang kosong, perlu diingat rmdir tidak dapat menghapus folder yang ada isinya.

mkdir folder1
rmdir folder1

rm

atau remove digunakan untuk menghapus sebuah file atau juga dapat menghapus folder yang ada isinya, bisa dengan menggunakan perintah rm dan ditambah opsi -r atau rekursif, perintah ini akan menghapus folder dan juga isi isinya.

rm [nama-file]
rm -r [nama-folder]

touch

digunakan untuk membuat sebuah file baru, misalnya kita ingin membuat file jadwalbelajar.txt, maka bisa gunakan perintah touch.

touch jadwalbelajar.txt

man, –help

man adalah singkatan manual, jadi kalau kita lagi bingung nih di salah satu command, kita bisa pake opsi ini untuk mendapatkan rincian penggunaan command tersebut.

man rm

cp

atau copy digunakan untuk menyalin sebuah file, file disalin dari jalur awal ke jalur yang kita inginkan.

cp [file-yang-mau-disalin] [tujuan]destination file]

mv

atau move digunakan untuk memindahkan sebuah file.

cp [file-yang-mau-dipindah] [tujuan]destination file]

locate

atau yang artinya menemukan digunakan untuk mencari sebuah file dalam terminal linux, misal kita ingin mencari file index, maka terminal akan menampilkan semua file yang mengandung kata index.

locate index

oke teman-teman, gimana soal terminal linux, seru kan, tentu masih banyak lagi yang bisa kita eksplor dari terminal linux ini.
