Menyalin File di Linux dari Satu Direktori ke Direktori Lain

Ferry Ananda Febian

Jan 9, 2023 • 1 min read

Di Linux, Anda dapat menyalin file dari satu direktori ke direktori lainnya, dengan memanfaatkan perintah cp atau copy.

Artikel ini akan membahas cara Menyalin File di Linux dari Satu Direktori ke Direktori Lain.

Berikut adalah sintaks cp untuk menyalin file dari satu direktori ke direktori lainnya:

cp [nama file yang akan disalin] [direktori tujuan]

Menyalin Salah Satu File

Misalnya, jika Anda ingin menyalin file bernama picture.jpg dari direktori saat ini ke direktori /home/user/Pictures, Anda dapat menjalankan perintah berikut:

cp picture.jpg /home/user/Pictures

Menyalin Beberapa File

Misalnya, jika Anda ingin menyalin beberapa file, (file1.txt, file2.txt) dari direktori saat ini ke direktori /home/user/Documents, Anda dapat menjalankan perintah berikut:

cp file1.txt file2.txt /home/user/Documents

Menyalin Semua Isi Direktori

Jika Anda ingin menyalin seluruh isi dari suatu direktori ke direktori lain, Anda dapat menambahkan flag -R (recursive). Misalnya, jika Anda ingin menyalin seluruh isi dari direktori /home/user/documents ke direktori /home/user/backup, Anda dapat menjalankan perintah berikut:

cp -R /home/user/Downloads /home/user/backup

Demikian artikel tentang cara Menyalin File di Linux dari Satu Direktori ke Direktori Lain, semoga bermanfaat.
