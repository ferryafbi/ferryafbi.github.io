Cara Upload File ke Github dengan Git

Ferry Ananda Febian

Nov 11, 2022 • 2 min read

Pengertian Git

Git adalah version control system yang memungkinkan tim untuk mengembangkan perangkat lunak secara kolaboratif atau bisa juga disebut manajemen kode perangkat lunak.

Tetapi sangat disarankan untuk memiliki repositori remote untuk tempat mengunduh dan mengupload kode.

Pengertian Github

Github adalah layanan host web bersama untuk proyek pengembangan perangkat lunak yang menggunakan sistem kendali versi Git dan layanan hosting internet. Pada artikel ini kita akan belajar cara upload file ke Github dengan bantuan Git.

Buat repository di Github

Langkah pertama adalah membuat repository atau simple nya sebut saja folder, di repository ini adalah kumpulan file-file yang akan diupload. Masukkan nama repository lalu klik buat repository.

Tampilan repository

Berikut adalah tampilan repository yang berhasil dibuat

Tentukan file project

Tentukan folder project yang akan diupload, sebagai contoh Saya akan menggunakan folder project-1 yang didalamnya ada file tetstaja.txt

Arahkan ke path

Masuk ke direktori project melalui terminal, disini Saya menggunakan terminal Linux, Anda bisa menyesuaikan di bagian opsi ini.

Masukkan command

git init
git add .
git commit -m "Initial commit"
git remote add origin <project url>
git push -u origin main

ssss

Setelah itu Anda akan diminta untuk memasukkan username dan password, gunakan username Github Anda, password bisa Anda dapatkan dengan melakukan generate di bagian settings > Developer Settings > Personal Access Tokens. Sangat disarankan untuk selalu memperbaharui tokens, Anda dapat menentukan masa kadaluarsa token yang dibuat.

Setelah mendapatkan token, salin lalu tempelkan di prompt password, 

jadi yang diinput pada bagian password bukanlah password akun Github, melainkan token yang di generate pada web Github yang mempunyai 36 karakter kurang lebih seperti ini: ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Cek repository di Github

Terlihat file dari repository lokal telah terupload di repository Github.

Oke, demikian artikel tentang cara upload file ke Github dengan bantuan Git, jika masih bingung langsung saja diskusi di kolom komentar.
