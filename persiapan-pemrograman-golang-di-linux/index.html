Persiapan Pemrograman Golang di Linux

Ferry Ananda Febian

Jan 9, 2023 • 1 min read

Untuk persiapan pemrograman dengan bahasa Golang di sistem Linux, Anda dapat mengunduh paket instalasi melalui website resmi Golang di https://go.dev/dl/ 

Ekstrak paket instalasi Golang yang telah diunduh. Biasanya terdapat dalam format tar.gz, sehingga dapat diekstrak menggunakan perintah tar -xvzf nama_paket.tar.gz.

Salin folder hasil ekstrak ke lokasi yang diinginkan. Biasanya Golang disimpan di /usr/local/go.

Tambahkan lokasi Go Lang ke dalam PATH sistem dengan menambahkan baris ini ke dalam file .bashrc di home directory:

export PATH=$PATH:/usr/local/go/bin
ShellScript

Muat ulang file .bashrc dengan perintah source ~/.bashrc.

source ~/.bashrc.
ShellScript

Verifikasi instalasi Golang dengan menjalankan perintah go version di terminal. Jika instalasi berhasil, maka akan muncul informasi versi Golang yang terinstall.

go version
ShellScript

Buat direktori kerja untuk proyek Golang kita. Biasanya direktori ini disebut GOPATH.

Tambahkan GOPATH ke dalam PATH sistem dengan menambahkan baris ini ke dalam file .bashrc di home directory:

export PATH=$PATH:$GOPATH/bin
ShellScript

Muat ulang file .bashrc dengan perintah source ~/.bashrc.

source ~/.bashrc
ShellScript

Setelah melakukan semua langkah di atas, kita sudah bisa mulai membuat proyek Golang di Linux. Kita dapat menggunakan perintah go build untuk membuat executable dari kode Golang kita, atau go run untuk langsung menjalankan kode Golang tanpa perlu membuat executable terlebih dahulu.
