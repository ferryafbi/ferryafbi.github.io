Cara Memperbaiki Error: Undefined Variable di PHP

Ferry Ananda Febian

Nov 22, 2022 • 1 min read

PHP adalah salah satu bahasa pemrograman website yang cukup populer hingga saat ini, tetapi disini kita melakukan coding, melainkan memperbaiki error yang ditandai dengan “Undefined Variable” di PHP.

Cara Memperbaiki Error: Undefined Variable in PHP

Ketika pesan error “Undefined Variable” tampil, hal ini menandakan bahwa PHP mencoba memproses variabel atau konstanta yang tidak terdefinisikan.

Contoh:

<?php 
$dob='2022-22-11'; 
echo $age; 
?>

Pada contoh diatas, Saya coba mendefinisikan variable $dob tetapi variabel yang dipanggil adalah $age.

Notice: Undefined variable: age in \test.php on line 3

Berikut dua cara memperbaiki pesan error tersebut:

Perbaiki Error dengan isset() function

Anda dapat mendefinisikan variabel secara global dan menggunakan isset() function untuk sekedar test jika variabel tersebut belum dipanggil.

Contoh:

<?php
global $age;
global $dob = '2021-01-01';
if(!isset($age)){
$age = 'Variable age is not set';
}

echo 'Age: ' . $age;
?>

Output

Variable age is not set

Perbaiki Error dengan setting variable sebagai blank atau default value

Solusi lain permasalahan ini adalah, dengan mengatur nya sebagai blank/kosong

<?php
$dob = '2021-01-01';
// Set Variable as Default
$age= isset($age) ? $age: '0';

echo 'Age: ' . $age;
?>

Output

Age: 0

Disini saya akan coba mengabaikan pemberitahuan fixed error, hal ini tidak disarankan. Berikut adalah cara menonaktifkan pesan error agar tidak muncul.

Nonaktifkan Pesan Error di File php.ini

Buka php.ini dengan teks editor. Perhatikan baris berikut ini:

error_reporting = E_ALL

Lalu ubah menjadi

error_reporting = E_ALL & ~E_NOTICE

Restart Apache server untuk menerapkan perubahan. Sekarang compiler PHP tidak lagi menampilkan psan error.

Nonaktifkan Pesan Error di File di Kode PHP 

Opsi ini bisa Anda lakukan jika tidak dapat mengakses file php.ini, tambahkan saja baris berikut di bagian paling atas kode PHP Anda.

<?php error_reporting (E_ALL ^ E_NOTICE); ?>
