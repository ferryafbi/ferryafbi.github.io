Comic Mischief - Project Pemrograman Golang untuk Pemula

Ferry Ananda Febian

Jan 9, 2023 • 2 min read

Project ini akan menampilkan informasi tentang sebuah comic, yaitu nama, harga, dan jumlah. Project ini juga menyediakan fungsi untuk menambah atau mengurangi jumlah comic, serta menghitung total harga comic tersebut. Project ini akan mencetak informasi tersebut ke dalam terminal setelah dijalankan.

Source Code

Berikut adalah source code project “Comic Mischief” dengan bahasa Golang:

package main

import "fmt"

// membuat struktur data "Comic" dengan field nama, harga, dan jumlah
type Comic struct {
nama string
harga int
jumlah int
}

// membuat fungsi dengan nama "tambah_comic" yang menerima sebuah parameter Comic c, dan mengembalikan sebuah Comic
func tambah_comic(c Comic) Comic {
c.jumlah++ // menambahkan 1 ke jumlah comic
return c // mengembalikan comic yang telah diupdate
}

// membuat fungsi dengan nama "kurangi_comic" yang menerima sebuah parameter Comic c, dan mengembalikan sebuah Comic
func kurangi_comic(c Comic) Comic {
if c.jumlah > 0 { // jika jumlah comic lebih besar dari 0
c.jumlah-- // mengurangi 1 dari jumlah comic
}
return c // mengembalikan comic yang telah diupdate
}

// membuat fungsi dengan nama "total_harga" yang menerima sebuah parameter Comic c, dan mengembalikan sebuah integer
func total_harga(c Comic) int {
return c.harga * c.jumlah // mengembalikan hasil perkalian harga dan jumlah comic
}

func main() {
// membuat sebuah comic dengan nama "Superman", harga 50.000, dan jumlah 5
comic := Comic{"Superman", 50000, 5}

fmt.Println("Comic:", comic.nama) // mencetak nama comic
fmt.Println("Harga:", comic.harga) // mencetak harga comic
fmt.Println("Jumlah:", comic.jumlah) // mencetak jumlah comic

// menambahkan 1 ke jumlah comic
comic = tambah_comic(comic)
fmt.Println("Jumlah setelah ditambah:", comic.jumlah) // mencetak jumlah comic setelah ditambah

// mengurangi 1 dari jumlah comic
comic = kurangi_comic(comic)
fmt.Println("Jumlah setelah dikurangi:", comic.jumlah) // mencetak jumlah comic setelah dikurangi

// mencetak total harga comic
fmt.Println("Total harga:", total_harga(comic))
}
Go
