Apa Itu RabbitMQ: Pengertian dan Cara Install di Ubuntu/Debian

Ferry Ananda Febian

Nov 3, 2022 • 1 min read

Pengertian RabbitMQ

RabbitMQ adalah broker pesan populer yang memungkinkan aplikasi dan layanan yang berbeda untuk berkomunikasi satu sama lain menggunakan antrian pesan. Pada artikel ini, kita akan belajar cara menginstal RabbitMQ di Ubuntu/Debian.

Di banyak industri seperti keuangan, aplikasi dan layanan yang memerlukan komunikasi satu sama lain secara asinkron. Jadi tools ini menggunakan antrian pesan di mana pengirim menambahkan data dan penerima terus melakukan polling secara teratur untuk mengambil pesan. RabbitMQ memudahkan pengelolaan beberapa antrian untuk membantu beberapa aplikasi & layanan berkomunikasi satu sama lain.

Cara Menginstal RabbitMQ di Ubuntu/Debian

Berikut adalah langkah-langkah untuk menginstal RabbitMQ di Ubuntu/Debian.

1. Update package

Buka terminal dan jalankan perintah apt-get update untuk memperbarui library linux.

sudo apt-get update
sudo apt-get -y upgrade

2. Aktifkan Repository RabbitMQ

Jalankan perintah berikut untuk mengaktifkan/enable pustaka RabbitMQ

sudo echo "deb http://www.rabbitmq.com/debian/ testing main" >> /etc/apt/sources.list

3. Tambahkan kunci verifikasi RabbitMQ

Jalankan perintah berikut untuk menambahkan kunci verifikasi

sudo curl http://www.rabbitmq.com/rabbitmq-signing-key-public.asc | sudo apt-key add -

Perintah diatas mungkin butuh beberapa menit untuk selesai, tergantung pada kecepatan internet Anda.

4. Update package

Update package Ubuntu/Debian sekali lagi

sudo apt-get update

5. Install RabbitMQ Server

Jalankan perintah berikut untuk install RabbitMQ Server.

sudo apt-get install rabbitmq-server

6. Aktifkan Rabbit Management Console

Rabbit management Console digunakan untuk mengelola RabbitMQ server, user dan antrian pesan secara efektif. Berikut perintah untuk mengaktifkannya.

sudo rabbitmq-plugins enable rabbitmq_management

Setelah berjalan, Anda dapat mengakses nya pada URL dibawah ini:

http://localhost:15672/

Ketika URL tersebut diakses. Anda akan diminta untuk mengisi form autentikasi, tenang saja, secara default username dan password adalah guest

Setelah berhasil login, Anda akan melihat tampilan dashboard RabbitMQ

Anda dapat gunakan dashboard untuk mengelola RabbitMQ server.

7. Kelola RabbitMQ

Berikut adalah beberapa perintah untuk mengelola RabbitMQ server.

#Aktifkan service:

service rabbitmq-server start

#Hentikan service:

service rabbitmq-server stop

#Mulai ulang service:

service rabbitmq-server restart

#cek status service:

service rabbitmq-server status

Kesimpulan

RabbitMQ adalah broker pesan yang banyak digunakan pada tingkat perusahaan, instalasi nya cukup mudah dan didukung dengan graphical interface mempermudah pengguna untuk mengoperasikannya. Konfigurasi defaultnya juga cukup baik untuk diterapkan pada lingkungan produksi.

Oke selesai, jadi itulah yang dimaksud dengan RabbitMQ dan cara konfigurasi di Ubuntu/Debian.
