Bekerja secara Lokal dengan Git

Ferry Ananda Febian

Jan 18, 2023 • 1 min read

Artikel ini akan membahas materi terkait Bekerja secara Lokal dengan Git dalam series belajar DevOps.

Sintaks Dasar Git

Untuk bekerja secara lokal dengan Git, pertama-tama Anda harus menginstall Git di komputer Anda. Setelah itu, Anda dapat membuat repository lokal dengan perintah “git init” pada direktori yang ingin Anda gunakan sebagai repository. 

git init
Git Commit

Kemudian, Anda dapat menambahkan file ke repository dengan perintah “git add” dan menyimpan perubahan dengan perintah “git commit”. 

#menambahkan file ke repo
git add

#menyimpan perubahan
git commit
Git Commit

Anda juga dapat melakukan operasi seperti mengambil perubahan dari repository remote dengan perintah “git pull” dan mengirim perubahan ke repository remote dengan perintah “git push”.

#mengambil perubahan dari remote repo
git pull

#mengirim perubahan
git push
Git Commit

Perlu diingat, perintah diatas masih sekedar sintaks dasar yang digunakan dalam pengoperasian Git, dalam praktik nya diperlukan penyesuaian terlebih dahulu.

Install Git di Linux

Untuk menginstall Git di sistem operasi Linux, Anda dapat mengikuti langkah-langkah berikut:

Buka terminal Anda.

Pastikan bahwa sistem Anda sudah diperbarui dengan perintah “sudo apt update”.

sudo apt update
ShellScript

Install Git dengan perintah “sudo apt-get install git”

sudo apt install git
ShellScript

Setelah proses instalasi selesai, Anda dapat mengecek versi Git yang diinstall dengan perintah “git –version atau git version”

git version
ShellScript

Selesai, Git sudah terinstall dan siap digunakan.
