Sintaks Dasar PHP

Ferry Ananda Febian

Nov 17, 2022 • 3 min read

Sintaks dasar PHP merupakan gaya atau aturan standar penulisan kode PHP, sebagai pemula, menguasai sintaks adalah suatu kewajiban. Berikut adalah sintaks dasar penulisan kode dalam PHP.

Tags PHP

Fungsi tags dalam PHP adalah untuk memberitahu dimana kode PHP yang akan diinterpretasikan. Tanda awal kode PHP adakah <?php dan diakhiri dengan ?>. Jika kode PHP ada didalam file khusus PHP, maka kurung PHP hanya sebagai opsional.

Contoh Tag Pembuka dan Penutup PHP

<!-- Tags 1 -->

<?php echo 'Script PHP seperti ini memungkinkan untuk disematkan pada file lain misal HTM dan XML'; ?>

<!-- Tags 2 -->

<?= 'print this string' ?>

<!-- kode diatas sama seperti -->

<?php echo 'print this string' ?>.

<!-- Tags 3 -->

<? echo 'kode ini menggunakan short tag, tetapi tetap berjalan'.

'jika short tag pembuka nya juga ditambahkan'; ?>

Tiga short tags diatas aktif secara default, Anda dapat menonaktifkannya di short_open_tag php.ini atau dinonaktifkan secara default jika PHP di konfigurasi dengan –disable-short-tags. Short tags mungkin tidak terlalu penting untuk dipahami, bagi pemula yang penting adalah mengerti mana kode PHP berdasarkan tags yang dibuat. Selain itu, dengan mengkonfigurasi short tags, dapat membuat kode tidak maksimal secara kompatibilitas.

Jika kode PHP dibuat dalam file khusus PHP, sebaiknya jangan gunakan penutup PHP, karena bisa saja file PHP Anda error hanya karena terdapat penambahan spasi dibawah tags penutup.

Contoh kode PHP tanpa menggunakan tags penutup.

Diluar dari HTML

Semua tags selain tags pembuka dan penutup PHP akan diabaikan, hal ini memungkinkan script PHP dapat disematkan di file lain seperti HTML dan XML. Biasanya teknik seperti ini digunakan untuk membuat template.

Contoh:

<p>Tags ini tidak akan parsed/urai oleh PHP dan akan ditampilkan di web</p>

<?php echo 'tags ini akan di parsed/urai oleh PHP'; ?>

<p>Tags ini juga tidak akan di parsed/urai oleh PHP</p>

Teknik ini bekerja sesuai yang diharapkan, karena ketika programmer membuat tags ?> maka kode PHP akan selesai sampai disitu. Simpelnya sebutan parsed adalah eksekusi atau kode yang berjalan.

Berikut contoh lainnya:

Menggunakan struktur dan kondisi

<?php if ($expression == true): ?>

kode ini akan ditampilkan kalau ekspresi tersebut bernilai benar

<?php else: ?>

kalau tidak, maka ini yang ditampilkan

<?php endif; ?>

Untuk mengeluarkan blok teks besar, keluar dari mode parsing PHP umumnya lebih efisien daripada mengirim semua teks melalui echo atau print.
