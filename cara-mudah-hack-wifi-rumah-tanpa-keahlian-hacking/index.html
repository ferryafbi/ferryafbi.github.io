Cara Mudah Hack WIFI Rumah Tanpa Keahlian Hacking

Ferry Ananda Febian

Nov 23, 2022 • 3 min read

Internet merupakan kebutuhan wajib di era digital saat ini, mulai dari anak-anak, remaja hingga orang dewasa. Sayang nya harga paket internet di Indonesia masih bisa dikatakan cukup mahal dan kecepatan internetnya pun lambat dibandingkan negara tetangga, sebut saja Malaysia dan Singapore. Di Beberapa kota di Indonesia sudah terjangkau koneksi internet fiber seperti Indihome, Xl Home, Icon+ dan lain-lain.

Cara Mudah Hack WIFI Rumah Tanpa Keahlian Hacking

Harap diingat, artikel ini tidak akan bertanggung jika Anda nantinya berurusan dengan hukum, sebaliknya, jadikan artikel ini sebagai panduan untuk lebih memproteksi perangkat dari ancaman hacking.

Langkah-Langkah Hack Password Wifi

Berikut adalah langkah-langkah hack password wifi:

Terkoneksi dengan wifi

Ada dua cara untuk terkoneksi dengan jaringan wifi:

Cara Normal

Cara normal adalah kondisi dimana perangkat yang Anda gunakan sudah pernah terkoneksi dengan jaringan wifi tersebut.

Cara Paksa

Cara paksa dilakukan ketika Anda tidak mengetahui password untuk login ke jaringan tersebut, ada banyak cara paksa yang bisa Anda lakukan:

  • Tanya ke pemilik wifi, lalu minta agar perangkat Anda terhubung ke jaringan tersebut
  • Menekan tombol wps

sebelum menekan tombol wps, pastikan wifi wps di perangkat Anda sudah aktif, dengan wps, Anda akan terkoneksi ke jaringan tanpa mengisi kata sandi. Cara kedua ini bisa Anda lakukan jika mempunyai akses ke router wifi. Jika targetnya adalah rumah tetangga, Anda bisa menyuruh anak kecil atau anggota keluarga target yang tidak terlalu mengerti untuk menekan tombol wps di router.

  • Menggunakan kabel ethernet

Sebagian besar router menyediakan port ethernet, Anda dapat menghubungkan perangkat ke jaringan dengan port tersebut. Cara ini juga hanya bisa dilakukan jika Anda mempunyai akses ke router wifi.

Login ke Router

Langkah selanjutnya adalah Login ke Router, langkah ini bisa dilakukan jika Anda sudah terhubung dengan jaringan wifi.

Akses panel router

Panel router wifi biasanya beralamat:

192.168.1.1

Harap diingat, link diatas hanya memuat tampilan login, Anda belum berhasil melakukan hacking.

Login User

Ini adalah fase penentu, jika Anda berhasil masuk ke router, Anda bebas melakukan konfigurasi apapun di router tersebut atau memasang backdoor untuk login dengan user dan password khusus yang tidak diketahui oleh target.

Untuk login user, Anda bisa coba menggunakan password default, berharap semoga belum diganti oleh si pemilik wifi tersebut. Berikut adalah user dan password login default dari berbagai perangkat, silahkan disesuaikan dengan perangkat target:

Login Huawei

Login Fiberhome

Login ZTE

Login Indihome (alternatif)

Berhasil Login

Setelah berhasil login, ada beberapa tahapan lagi yang harus Anda lakukan seperti buat SSID hide dan membuat user login baru sebagai backdoor.

Buat SSID Hide

SSID atau Service Set Identifier hide adalah nama jaringan yang dipancarkan oleh router wifi yang bersifat tersembunyi, artinya SSID ini hanya bisa diakses jika user mengetahui nama SSID dan password. Untuk cara pembuatan SSID di tiap perangkat wifi berbeda-beda, tidak saya jelaskan secara rinci karena akan membuat artikel ini terlalu panjang, silahkan googling untuk proses ini.

Buat User Baru

User baru ini akan dimanfaatkan jika user default sudah tidak bisa digunakan lagi atau passwordnya sudah diganti oleh si pemilik wifi. Sama seperti membuat SSID hide, buat user baru di tiap perangkat akan berbeda-beda. Silahkan googling untuk proses ini.
