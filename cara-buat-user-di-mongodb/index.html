Cara Buat User di MongoDB

Ferry Ananda Febian

Nov 22, 2022 • 1 min read

MongoDB adalah database NoSQL yang sangat populer dikalangan developer, MongoDB memungkinkan Anda untuk menyimpan berbagai tipe data secara bersamaan dengan menggunakan dokumen JSON. Struktur primary Key nya memudahkan untuk mencari dan mengambil nilai dari database MongoDB. 

Artikel ini akan membahas cara membuat user/pengguna serta menetapkan role nya di MongoDB.

Buat User di MongoDB

Masuk ke database MongoDB:

sudo mongodb

Masukkan username dan password login MongoDB.

Selanjutnya Anda dapat membuat user baru dengan function CreateUser, seperti berikut ini:

db.createUser(
{ user: "ini_user_1",
pwd: "ini_password_user_1",
roles:[{role: "userAdmin" , db:"admin"]})

Perhatikan kode diatas, Saya membuat user baru dengan nama (ini_user_1) dan password (ini_password_user_1).

Tidak lupa juga menambahkan role atau peran dari user tersebut, Saya menambahkan role sebagai userAdmin, role ini mengizinkan user untuk untuk menjadi administrator database tertentu. Serta menambahkan lokasi database untuk user tersebut di database admin. Berikut daftar semua peran pengguna yang tersedia di Mongodb, Database Administration Roles.

Berikut adalah output hasil eksekusi kode diatas:

Successfully added user: user: {
"ini_user_1",
roles:[{role: "userAdmin" , db:"admin"]
}
