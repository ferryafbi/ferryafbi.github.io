Cara Grep Lebih dari Satu Strings di Linux

Ferry Ananda Febian

Nov 22, 2022 • 2 min read

Grep adalah tools yang ampuh untuk mencari string, pola, dan kata dalam file. Hal ini memungkinkan Anda untuk mencari satu atau lebih strings serta pola berdasarkan kebutuhan. 

Artikel ini akan membahas bagaimana cara menggunakan grep untuk mencari lebih dari satu strings di Linux.

Cara Grep Lebih dari Satu Strings di Linux

Berikut ini adalah sintaks dasar grep untuk mencari lebih dari satu strings dan patterns atau pola

sudo grep [options] 'pattern1|pattern2|...' /path/to/file

Atau

command | grep [options] 'pattern1|pattern2|...'

Pada perintah pertama, grep diminta untuk mencari beberapa pola (pattern1, pattern2, …) dalam sebuah file sementara dalam kasus kedua, output diteruskan dari perintah lain ke grep sehingga dapat mencari string yang diperlukan.

Dalam kedua kasus, grep akan menampilkan semua baris yang berisi salah satu pola yang terdapat dalam file.

Cara Mencari Banyak String dalam sebuah file

Misalnya Anda ingin mencari banyak string (air, matahari) di file /home/ubuntu/poem.txt. Berikut perintahnya:

sudo grep 'air|matahari' /home/ubuntu/fileaja.txt

Jika Anda tidak menentukan alamat file secara lengkap, grep akan mencari file tersebut di direktori Anda saat ini.

Jika Anda ingin mencari 3 string (air, matahari, bulan, tambahkan saja ke pola pencarian Anda seperti yang ditunjukkan

sudo grep 'air|matahari|bulan' /home/ubuntu/fileaja.txt

Perintah grep di atas akan mencari kecocokan hanya untuk salah satu dari string yang ditentukan.

Jika Anda ingin mencari pattern/pola string tertentu, gunakan opsi -E. Opsi ini akan menganggap strings sebagai ekspresi reguler dan memungkinkan Anda untuk menyertakan karakter regex seperti *, ?, {}, dll.

sudo grep -E 'water|sun|moon' /home/ubuntu/poem.txt

Atau

sudo grep -e water -e sun -e moon /home/ubuntu/ini-file.txt

Cari Dengan Pencocokan Persis

Jika Anda ingin mencari strings yang sama persis, gunakan opsi -w.

sudo grep -w 'air|matahari' /home/ubuntu/ini-file.txt

Abaikan Case Saat Mencari String

Jika Anda ingin mengabaikan huruf besar-kecil saat mencari beberapa string dan pola, gunakan opsi -i.

sudo grep -i 'air|matahari' /home/ubuntu/ini-file.txt

Dapatkan Jumlah Strings

Jika Anda ingin menghitung berapa banyak string dalam pencarian sebuah file, gunakan opsi -c.

sudo grep -c 'air|matahari’ /home/ubuntu/ini-file.txt

Cari banyak pola di banyak file

Jika Anda ingin mencari banyak string dalam banyak file, Anda dapat menggunakan karakter wildcard untuk menyebutkan jenis file tertentu (misal. File *.txt)

sudo grep 'air|matahari' /home/ubuntu/*.txt

Perintah di atas hanya akan mencari string yang ditentukan di semua file .txt di direktori /home/ubuntu tetapi tidak di sub direktori mana pun. Jika Anda juga ingin mencari banyak string dalam file yang berbeda di subdirektori, gunakan opsi -R untuk melakukan pencarian rekursif.

sudo grep -R 'water|sun' /home/ubuntu/*.txt

Perintah di atas juga akan mencari string di file .txt yang terdapat di semua sub folder /home/ubuntu.
