Belajar React Dasar | List

Ferry Ananda Febian

Mar 7, 2023 • 2 min read

List sangat penting dalam hal mengembangkan UI situs web. Terutama sering digunakan untuk menampilkan menu di situs web, misalnya menu navbar. Pada JavaScript biasa, kita dapat menggunakan array untuk membuat list. Kita dapat membuat list di React dengan cara yang sama seperti yang dilakukan pada javascript biasa.

Selanjutnya artikel ini akan membahas lebih dalam terkait list pada React.

Pengertian

List pada React adalah salah satu tipe data yang digunakan untuk menyimpan kumpulan nilai atau elemen dalam sebuah variabel. List juga bisa disebut sebagai daftar atau array.

Cara Membuat List

Untuk membuat list pada React, kita dapat menggunakan sintaksis yang sama seperti di JavaScript, yaitu dengan menempatkan elemen-elemen atau nilai-nilai yang ingin kita simpan di dalam tanda kurung siku ([]). Kita juga dapat menggunakan method-method seperti map(), filter(), dan reduce() untuk memanipulasi data di dalam list tersebut.

Berikut adalah contoh fungsi JavaScript map() untuk menelusuri list:

var nomor = [1, 2, 3, 4, 5];
const updateNomor = nomor.map((number) => {
  return number + 2;
});

console.log(updateNomor);
JavaScript

Kode diatas akan menghasilkan output seperti di bawah ini:

[3, 4, 5, 6, 7]

Selanjutnya kita akan coba menampilkan kumpulan data dalam bentuk list atau daftar di dalam komponen React. Dalam React, rendering list biasanya dilakukan dengan menggunakan konsep “pengulangan” atau “looping”, yang memungkinkan kita untuk menampilkan setiap elemen dalam list secara otomatis.

import React from "react";
import ReactDOM from "react-dom";

function Navmenu(props) {
  const list = props.menuitems;
  const updatedList = list.map((listItems) => {
    return <li>{listItems}</li>;
  });
  return <ul>{updatedList}</ul>;
}

const menuItems = [1, 2, 3, 4, 5];

ReactDOM.render(<Navmenu menuitems={menuItems} />, document.getElementById("root"));
JavaScript

Output yang dihasilkan:

Bisa dilihat pada output di atas, unordered list berhasil di render ke browser tetapi pesan warning dicatat ke konsol.

Peringatan: Setiap child dalam larik atau iterator harus memiliki penyangga “key” yang unik

Pesan warning di atas mengatakan bahwa setiap item daftar dalam daftar unordered harus memiliki kunci yang unik. “Kunci” adalah atribut string khusus yang perlu disertakan saat membuat daftar elemen di React. Materi tentang key akan dibahas secara rinci di artikel selanjutnya.

Berikut adalah contoh kode React yang sudah diperbaharui dengan key:

import React from "react";
import ReactDOM from "react-dom";

function Navmenu(props) {
  const list = props.menuitems;
  const updatedList = list.map((listItems) => {
    return <li key={listItems.toString()}>{listItems}</li>;
  });

  return <ul>{updatedList}</ul>;
}

const menuItems = [1, 2, 3, 4, 5];

ReactDOM.render(<Navmenu menuitems={menuItems} />, document.getElementById("root"));
JavaScript

Kode diatas akan memberikan output yang sama dengan kode sebelumnya tetapi kali ini tanpa peringatan apapun. Key digunakan dalam React untuk mengidentifikasi item mana dalam daftar yang diubah, diperbarui, atau dihapus. Dengan kata lain, key dapat digunakan untuk memberi identitas pada elemen dalam daftar.
