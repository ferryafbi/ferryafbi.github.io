Rangkuman Materi CCNA-1: Configure IP Addressing

Ferry Ananda Febian

Jan 3, 2023 • 2 min read

Konfigurasi IP (Internet Protocol) adalah suatu proses yang dilakukan untuk mengatur atau mengkonfigurasi alamat IP yang akan digunakan oleh komputer atau perangkat lain yang terhubung ke jaringan. Alamat IP adalah suatu angka yang unik yang digunakan untuk mengidentifikasi setiap perangkat yang terhubung ke jaringan Internet atau jaringan lokal (LAN). Konfigurasi IP juga meliputi penentuan subnet mask dan gateway default yang akan digunakan oleh perangkat tersebut. Subnet mask digunakan untuk menentukan bagian dari alamat IP yang menyatakan jaringan, sementara gateway default adalah alamat IP dari perangkat yang digunakan untuk menghubungkan jaringan lokal dengan jaringan yang lebih besar, seperti Internet. Konfigurasi IP sangat penting karena memungkinkan perangkat untuk terhubung ke jaringan dan berkomunikasi dengan perangkat lain di jaringan tersebut.

Manual IP Address Configuration

IP Address Configuration manual adalah proses menentukan secara manual alamat IP yang akan digunakan oleh perangkat yang terhubung ke jaringan. Anda harus menentukan alamat IP, subnet mask, dan default gateway secara manual untuk setiap perangkat yang terhubung ke jaringan.

Contoh konfigurasi IP address secara manual

Automatic IP Address Configuration

IP Address Configuration automatic adalah proses menentukan alamat IP secara otomatis oleh perangkat yang terhubung ke jaringan. Perangkat akan menggunakan protokol seperti Dynamic Host Configuration Protocol (DHCP) atau Automatic Private IP Addressing (APIPA) untuk mendapatkan alamat IP, subnet mask, dan default gateway secara otomatis dari server DHCP atau APIPA.

Syntax Checker – Verify Windows PC IP Configuration

Syntax checker adalah sebuah alat yang digunakan untuk memeriksa apakah sintaks (struktur dan penulisan) dari suatu kode atau perintah yang ditulis sesuai dengan standar yang telah ditetapkan. Pada sistem operasi Windows, ada beberapa cara yang dapat dilakukan untuk memeriksa konfigurasi IP (Internet Protocol) komputer, salah satunya dengan menggunakan perintah “ipconfig”. Perintah ini akan menampilkan informasi tentang konfigurasi jaringan yang sedang digunakan oleh komputer, termasuk alamat IP, subnet mask, dan gateway default. Syntax checker dapat digunakan untuk memeriksa apakah perintah “ipconfig” yang Anda masukkan benar dan sesuai dengan standar yang telah ditetapkan.

Switch Virtual Interface Configuration

Virtual interfaces (VLAN interfaces) pada perangkat Cisco adalah logis interface yang dapat dibuat pada switch untuk membagi satu switch menjadi beberapa bagian virtual yang masing-masing dapat di konfigurasi dengan VLAN ID yang berbeda. Ini berguna jika Anda ingin memisahkan jaringan Anda ke dalam bagian yang lebih kecil untuk keamanan atau pengelolaan yang lebih baik.
