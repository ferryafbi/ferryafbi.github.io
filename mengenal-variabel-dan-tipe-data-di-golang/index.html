Mengenal Variabel dan Tipe Data di Golang

Ferry Ananda Febian

Jan 9, 2023 • 3 min read

Di sesi ini, kita akan mengenal variabel dan tipe data di Golang

Apa Itu Variabel

Variabel adalah tempat yang digunakan untuk menyimpan nilai-nilai di dalam suatu program. Dalam Go, Anda perlu mendeklarasikan variabel sebelum dapat menggunakannya dalam program Anda. Anda dapat mendeklarasikan variabel dengan menggunakan kata kunci “var” atau dengan menggunakan nilai default yang diberikan oleh Go. Contohnya:

// mendeklarasikan variabel dengan menggunakan kata kunci "var"
var a int = 10
var b string = "hello"

// mendeklarasikan variabel dengan menggunakan nilai default yang diberikan oleh Go
var c = 20
var d = "world"
Go

Selain itu Variabel, Go memiliki beberapa tipe data built-in seperti integer, float, boolean, dan string. Anda juga dapat membuat tipe data sendiri dengan menggunakan tipe struct atau menggunakan tipe interface untuk membuat tipe data yang lebih abstrak.

Tipe Data Integer

Tipe data integer di Go adalah tipe data yang digunakan untuk menyimpan bilangan bulat. Ada beberapa tipe integer built-in yang tersedia di Go, yaitu:

  • int: tipe integer dengan panjang yang sesuai dengan arsitektur komputer yang sedang digunakan (32-bit atau 64-bit)
  • int8: tipe integer dengan panjang 8-bit
  • int16: tipe integer dengan panjang 16-bit
  • int32: tipe integer dengan panjang 32-bit
  • int64: tipe integer dengan panjang 64-bit

Anda dapat mendeklarasikan variabel dengan tipe integer dengan cara seperti ini:

var a int = 10
var b int8 = 100
var c int16 = 1000
var d int32 = 10000
var e int64 = 100000
Go

Anda juga dapat menggunakan notasi shorthand untuk mendeklarasikan variabel dengan tipe integer seperti ini:

a := 10
b := int8(100)
c := int16(1000)
d := int32(10000)
e := int64(100000)
Go

Tipe Data Float

Tipe data float adalah tipe data yang digunakan untuk menyimpan nilai-nilai desimal atau bilangan koma flotan. Dalam Go, tipe data float terbagi menjadi dua yaitu float32 dan float64. Tipe data float32 memiliki presisi sebanyak 32 bit, sedangkan tipe data float64 memiliki presisi sebanyak 64 bit.

Contoh penggunaan tipe data float dalam Go:

var a float32 = 3.14
var b float64 = 3.14159265359
Go

Perbedaan utama antara tipe data float32 dan float64 adalah tingkat presisinya. Tipe data float32 memiliki presisi yang lebih rendah dibandingkan dengan tipe data float64, sehingga biasanya lebih cocok digunakan untuk aplikasi yang membutuhkan kinerja yang lebih tinggi. Namun, tipe data float64 lebih cocok untuk aplikasi yang membutuhkan presisi yang lebih tinggi, meskipun mungkin akan mengorbankan kinerja sedikit.

Tipe Data Boolean

Tipe data boolean di Go adalah tipe data yang digunakan untuk menyimpan nilai true atau false. Tipe data boolean biasanya digunakan dalam pernyataan kondisi untuk menentukan apakah suatu kondisi terpenuhi atau tidak. Contohnya:

// mendeklarasikan variabel boolean
var a bool = true
var b bool = false

if a {
    fmt.Println("a is true")
}

if !b {
    fmt.Println("b is false")
}
Go

Dalam contoh diatas, variabel a akan dianggap sebagai benar karena memiliki nilai true, sedangkan variabel b akan dianggap sebagai salah karena memiliki nilai false.

Tipe Data String

Di bahasa pemrograman Go, tipe data string adalah sebuah tipe data yang menyimpan sekumpulan karakter. String merupakan tipe data yang bersifat immutable, artinya setelah sebuah string dibuat, tidak dapat diubah atau dimodifikasi lagi. Anda dapat membuat sebuah string dengan menggunakan tanda petik ganda (” “) atau tanda petik satu (‘ ‘).

Contoh:

package main

import "fmt"

func main() {

// Menggunakan tanda petik ganda
s1 := "Hello, world!"

// Menggunakan tanda petik satu
s2 := 'Hello, world!' // akan menyebabkan error karena tanda petik satu hanya bisa digunakan untuk menyimpan satu karakter saja

}
Go

Tipe Data Struct

Tipe data struct di Golang adalah tipe data yang memungkinkan pengguna untuk membuat struktur data dengan mengkombinasikan tipe data lainnya menjadi satu unit. Struct bisa terdiri dari tipe data seperti integer, string, boolean, dan lain-lain. Struktur data ini bisa digunakan untuk menyimpan data yang terkait dengan satu entitas, seperti data pelanggan, data barang, dan lain-lain. Struktur data juga bisa digunakan untuk menyimpan data yang memiliki relasi dengan tipe data lainnya, seperti menyimpan data pelanggan bersama dengan data pemesanan yang dilakukan oleh pelanggan tersebut.

contoh:

// tipe data struct
type Point struct {

    x int
    y int

}

var p Point = Point{1, 2}
Go

Tipe Data Interface

Tipe data interface di Golang adalah tipe data yang menggambarkan set atau kumpulan method (fungsi) yang harus diimplementasikan oleh tipe data lain. Tipe data interface tidak memiliki implementasi methodnya sendiri, tetapi menyatakan bahwa tipe data lain harus memiliki method yang sesuai dengan tipe data interface tersebut untuk dapat menggunakannya. Contoh penggunaan tipe data interface adalah ketika kita ingin membuat sebuah tipe data yang harus memiliki method read dan write, maka kita dapat membuat tipe data interface dengan nama “ReaderWriter” yang menyatakan bahwa tipe data lain harus memiliki method Read() dan Write() untuk dapat mengimplementasikan tipe data interface tersebut.

Contoh:

// tipe data interface
type Shape interface {

    area() float64

}
Go
