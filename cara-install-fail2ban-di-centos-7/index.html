Cara Install Fail2ban di CentOS 7

Ferry Ananda Febian

Jan 9, 2023 • 1 min read

Fail2ban adalah software open source yang digunakan untuk memblokir IP yang melakukan percobaan login yang gagal ke sistem. Aplikasi ini bekerja dengan memindai log sistem dan mengeluarkan IP yang melakukan percobaan login yang gagal sebanyak maksimum yang diizinkan dalam waktu tertentu. Hal ini berguna untuk mencegah serangan brute-force pada sistem. Fail2ban tersedia untuk sistem operasi seperti Linux, BSD, dan macOS.

Berikut langkah-langkah Cara Install Fail2ban di CentOS 7:

Install repositori epel

epel atau (Extra Packages for Enterprise Linux)

sudo yum install epel-release
ShellScript

Instal Fail2ban

sudo yum install fail2ban
ShellScript

Salin file konfigurasi

Salin file konfigurasi default Fail2ban ke direktori konfigurasi fail2ban dengan perintah:

sudo cp /etc/fail2ban/jail.conf /etc/fail2ban/jail.local
ShellScript

Konfigurasi Fail2ban (opsional)

Buka file jail.local dengan editor teks seperti nano atau vi:

sudo nano /etc/fail2ban/jail.local
ShellScript

Pada bagian [DEFAULT], Anda bisa mengubah parameter seperti “ignoreip“, “bantime“, dan “maxretry” sesuai kebutuhan Anda.

  • Parameter “ignoreip” menentukan IP yang tidak akan diblokir oleh Fail2ban
  • “bantime” menentukan berapa lama seorang pengguna akan diblokir jika melakukan percobaan login yang gagal
  • “maxretry” menentukan berapa kali seorang pengguna diizinkan untuk melakukan percobaan login yang gagal sebelum diblokir.

Di bawah [DEFAULT], Anda bisa menambahkan konfigurasi untuk layanan yang ingin Anda proteksi dengan Fail2ban. Misalnya, jika Anda ingin melindungi layanan SSH, Anda bisa menambahkan blok konfigurasi berikut:

[ssh]
enabled = true
port    = ssh
filter  = sshd
logpath = /var/log/secure
ShellScript

Mulai Fail2ban

Mulai Fail2ban dengan perintah:

sudo systemctl start fail2ban
ShellScript

Setelah Fail2ban dijalankan, pastikan bahwa Fail2ban akan dijalankan secara otomatis saat sistem melakukn boot dengan perintah:

sudo systemctl enable fail2ban
ShellScript

Anda bisa mengecek status Fail2ban dengan perintah:

sudo fail2ban-client status
ShellScript

Oke selesai, sekarang Fail2ban sudah terinstall dan dijalankan di sistem CentoS. Demikian artikel tentang Cara Install Fail2ban di CentOS 7, semoga bermanfaat.
