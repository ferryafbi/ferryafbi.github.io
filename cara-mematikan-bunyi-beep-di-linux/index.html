Cara Mematikan Bunyi Beep di Linux

Ferry Ananda Febian

Jan 18, 2023 • 1 min read

Berikut adalah tutorial Cara Mematikan Bunyi Beep di Linux

Bunyi Beep di Linux bagi saya sendiri atau sebagian dari teman-teman mungkin merasa terganggu, karena bunyi beep nya muncul secara tiba-tiba dengan suara yang lumayan cukup keras.

Bagi user Linux yang masih pemula mungkin bingung untuk mengkonfigurasi agar bunyi beep tidak muncul atau dimatikan.

Untuk mematikan suara beep di Linux, kita bisa mengkonfigurasi nya di file /etc/inputrc.

Masuk ke file /etc/inputrc

sudo vim /etc/inputrc
ShellScript

jika Anda tidak familiar dengan vim, bisa juga menggunakan nano

sudo nano /etc/inputrc
ShellScript

setelah masuk ke dalam file, cari baris dengan tulisan 

# set bell-style none

lalu hapus tanda komentar (#) untuk mengaktifkan baris tersebut.

Setelah itu simpan dan keluar.

Cara Kedua

Jika cara yang pertama tidak berhasil, Anda bisa coba menonaktifkan modul pcspk dengan perintah berikut ini.

sudo rmmod pcspkr ; echo "blacklist pcspkr" >>/etc/modprobe.d/blacklist.conf
ShellScript

Demikian tutorial Cara Mematikan Bunyi Beep di Linux, semoga bermanfaat.
