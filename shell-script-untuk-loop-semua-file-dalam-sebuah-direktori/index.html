Shell Script untuk Loop Semua File dalam Sebuah Direktori

Ferry Ananda Febian

Jan 9, 2023 • 1 min read

Dengan Shell Script, kita dapat melakukan Loop Semua File dalam Sebuah Direktori.

Untuk melakukan loop melalui semua file dalam sebuah direktori, kamu dapat menggunakan perintah for dalam shell script. Berikut ini adalah contoh sederhananya:

#!/bin/bash

# Iterate through all files in the directory
for file in /path/to/directory/*
do
  # Process each file
  echo "Processing $file"
done
ShellScript

Perintah for akan mengeloop melalui setiap file dalam direktori /path/to/directory/, dan mengeksekusi blok kode di dalam loop untuk setiap file tersebut. Dalam contoh di atas, kita hanya mencetak nama file yang sedang diproses, tetapi Anda dapat menambahkan kode lain di dalam loop untuk memproses file tersebut sesuai kebutuhan.

Jika Anda ingin melakukan loop hanya pada file-file yang memenuhi kriteria tertentu, kamu dapat menggunakan wildcard (*) dan perintah if di dalam loop untuk mengecek apakah file tersebut memenuhi kriteria tersebut. Sebagai contoh:

#!/bin/bash

# Iterate through all files in the directory
for file in /path/to/directory/*
do
  # Check if the file is a regular file (not a directory or other special file)
  if [ -f "$file" ]
  then
    # Process the file if it is a regular file
    echo "Processing regular file: $file"
  fi
done
ShellScript

Di sini, kita mengecek apakah file yang sedang diproses adalah file biasa (bukan direktori atau file khusus lainnya) dengan menggunakan perintah if dan operator [ -f “$file” ]. Jika file tersebut adalah file biasa, kita akan memproses file tersebut dengan mencetak pesan di layar. Kamu dapat menambahkan kode lain di dalam blok if untuk memproses file tersebut sesuai kebutuhanmu.
