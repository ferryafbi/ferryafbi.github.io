Mengenal Fungsi di Golang

Ferry Ananda Febian

Jan 9, 2023 • 2 min read

Di sesi ini, kita akan belajar apa itu fungsi dalam bahasa pemrograman Golang.

Fungsi di Golang

Fungsi di Golang adalah sekumpulan pernyataan yang digabungkan menjadi satu unit yang dapat dipanggil dengan menggunakan nama yang diberikan. Fungsi dapat menerima input yang disebut parameter dan dapat mengembalikan output yang disebut return value.

Contoh penggunaan fungsi di Golang:

// membuat fungsi dengan nama "tambah" yang menerima dua parameter integer a dan b, dan mengembalikan sebuah integer
func tambah(a int, b int) int {

// pernyataan yang akan dijalankan saat fungsi dipanggil
return a + b // mengembalikan hasil penjumlahan a dan b

}

// memanggil fungsi "tambah" dengan mengirimkan dua integer sebagai parameter, dan menyimpan hasil return value ke dalam variabel c
c := tambah(1, 2)
Go

Fungsi juga dapat tidak menerima parameter atau tidak mengembalikan return value. Contohnya:

// membuat fungsi dengan nama "hello" yang tidak menerima parameter dan tidak mengembalikan return value
func hello() {

fmt.Println("Hello, World!") // mencetak string "Hello, World!"
}

// memanggil fungsi "hello" tanpa mengirimkan parameter
hello()

// membuat fungsi dengan nama "pangkat" yang menerima sebuah parameter integer x, dan mengembalikan sebuah integer

func pangkat(x int) int {
return x * x // mengembalikan hasil pangkat x

}

// memanggil fungsi "pangkat" dengan mengirimkan sebuah integer sebagai parameter, dan tidak menyimpan hasil return value
pangkat(2)
Go

Fungsi juga dapat menerima lebih dari satu return value. Contohnya:

// membuat fungsi dengan nama "kali" yang menerima dua parameter integer a dan b, dan mengembalikan dua return value integer
func kali(a int, b int) (int, int) {

return a * b, a + b // mengembalikan hasil perkalian a dan b, dan hasil penjumlahan a dan b

}

// memanggil fungsi "kali" dengan mengirimkan dua integer sebagai parameter, dan menyimpan hasil return value ke dalam variabel c dan d

c, d := kali(2, 3)
Go

Fungsi juga dapat memiliki parameter dengan tipe data yang sama, sehingga tidak perlu menuliskan tipe data untuk setiap parameter. Contohnya:

// membuat fungsi dengan nama "tambah_tiga" yang menerima tiga parameter integer a, b, dan c, dan mengembalikan sebuah integer
func tambah_tiga(a, b, c int) int {
return a + b + c // mengembalikan hasil penjumlahan a, b, dan c
}

// memanggil fungsi "tambah_tiga" dengan mengirimkan tiga integer sebagai parameter, dan menyimpan hasil return value ke dalam variabel d
d := tambah_tiga(1, 2, 3)
Go

Fungsi juga dapat memiliki parameter dengan tipe data yang berbeda, dan dapat memiliki nama yang berbeda untuk setiap parameter. Contohnya:

// membuat fungsi dengan nama "luas_persegi" yang menerima dua parameter integer panjang dan lebar, dan mengembalikan sebuah integer
func luas_persegi(panjang, lebar int) int {
return panjang * lebar // mengembalikan hasil perkalian panjang dan lebar
}

// memanggil fungsi "luas_persegi" dengan mengirimkan dua integer sebagai parameter, dan menyimpan hasil return value ke dalam variabel e
e := luas_persegi(4, 5)
Go

Kesimpulan Fungsi di Golang

Fungsi di Golang adalah sekumpulan pernyataan yang digabungkan menjadi satu unit yang dapat dipanggil dengan menggunakan nama yang diberikan. Fungsi dapat menerima input yang disebut parameter dan dapat mengembalikan output yang disebut return value. Fungsi dapat tidak menerima parameter atau tidak mengembalikan return value, atau bahkan dapat menerima lebih dari satu return value. Fungsi juga dapat memiliki parameter dengan tipe data yang sama atau berbeda, dan dapat memiliki nama yang berbeda untuk setiap parameter. Penggunaan fungsi dapat membantu mempercantik kode dan memudahkan pengembangan aplikasi.
