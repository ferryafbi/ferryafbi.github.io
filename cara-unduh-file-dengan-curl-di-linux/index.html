Cara Unduh File dengan Curl di Linux

Ferry Ananda Febian

Nov 23, 2022 • 2 min read

cURL adalah tools yang digunakan untuk mentransfer data dan file melalui jaringan. cURL mendukung banyak protokol seperti HTTP, HTTPS, FTP, SFTP, dan lain-lain.

Artikel ini akan membahas bagaimana cara mengunduh file dengan cURL di Linux.

Cara Unduh File dengan Curl di Linux

1. Install cURL di Linux

Sebagian besar distro Linux sudah menyediakan cURL secara default, namun jika tidak tersedia silahkan melakukan instalasi sesuai distro yang Anda gunakan menggunakan terminal:

## Debian/Ubuntu

sudo apt install curl

## Fedora/CentOS/RHEL

sudo dnf install curl

## OpenSUSE

sudo zypper install curl

2. Cek Versi Curl

curl --version

3. Unduh File dengan cURL

Ini adalah sintaks dasar cURL:

curl [options] url_with_protocol

Ketika mengunduh file dengan cURL, Anda perlu menuliskan spesifik alamat URL beserta protokol yang digunakan, contoh:

sudo curl --output your_file.pdf https://www.example.com/my_file.pdf 

Atau

sudo curl -o your_file.pdf https://www.example.com/my_file.pdf

Jika Anda tidak menentukan opsi -o atau –output, curl akan menyimpan file yang diunduh dengan nama acak/random.

Jika Anda ingin menyimpan file yang diunduh dengan nama yang sama, gunakan opsi -O (kapital O).

sudo curl -O https://www.example.com/my_file.pdf

4. Resume atau pause Unduhan Gagal

Karena alasan tertentu unduhan dapat terhenti, maka Anda dapat melanjutkan unduhan file dengan perintah -C

sudo curl -C - -O https://www.example.com/my_file.pdf

5. Unduh File dengan URL Redirect

Secara default, curl tidak mengunduh file dari URL yang redirect atau dialihkan. Anda dapat mengaturnya dengan menggunakan opsi -L.

sudo curl -L -O https://www.example.com/my_file.pdf

6. Unduh Banyak File dengan cURL

Jika Anda ingin mengunduh banyak file dari URL yang berbeda maka Anda perlu menambahkan setiap URL secara terpisah dengan opsi -O di setiap URL file.

sudo curl -O url1 -O url2
sudo curl -O https://www.example.com/my_file.pdf \
                  -O https://www.example.com/my_file2.pdf

Anda juga dapat mengunduh file yang URL disimpan disebuah file, misal (urls.txt) maka Anda dapat meminta curl untuk memilih URL tersebut dari file lalu mengunduhnya dengan perintah -n. Pastikan Anda mencantumkan 1 URL di setiap baris file.

URLs.txt
------
https://www.example.com/my_file.pdf

https://www.example.com/my_file2.pdf

https://www.example.com/my_file3.pdf

Berikut adalah perintah untuk mengunduh daftar URL dari file. Pada kasus ini perlu menggunakan perintah xargs.

xargs -n 1 curl -O < "urls.txt"

7. Unduh File yang Diproteksi Password

Jika file yang diunduh di proteksi dengan password, Anda dapat mengunduhnya dengan menambahkan opsi pengguna & kata sandi dengan opsi -u. Ganti test_user & test_password di bawah ini dengan nama pengguna & kata sandi Anda.

curl -u test_user:test_password https://www.example.com/my_file.pdf

8. Unduh File dengan Proxy Server

Berikut adalah sintaks untuk mengunduh file dengan proxy server:

curl -x proxy-server-ip:PORT -O url

Jika diperlukan autentikasi, cukup tambahkan opsi username dan password dengan sintaks berikut ini:

curl -u username:password -x proxy-server-ip:PORT -O url

Contoh:

curl -x 'https://proxy.example.com' -v -O https://www.example.com/my_file.pdf

9. Rate Limit Unduhan

Anda juga dapat membatasi kecepatan unduhan dengan menambahkan opsi –limit-rate seperti berikut ini:

curl --limit-rate {speed} url

Contoh limit unduhan dengan kecepat 300kb/detik

sudo curl --limit-rate 300k -O https://www.example.com/my_file.pdf
