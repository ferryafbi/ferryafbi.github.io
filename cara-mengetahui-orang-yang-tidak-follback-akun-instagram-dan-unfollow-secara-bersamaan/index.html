Cara Mengetahui Orang yang Tidak Follback Akun Instagram dan Unfollow Secara Bersamaan

Ferry Ananda Febian

Mar 6, 2023 • 2 min read

Oke gan, mungkin hampir semua orang sudah pernah mengalami atau ingin tau, siapa aja sih orang yang tidak memfollback akun instagram kita. Apapun alasan kalian mengunjungi artikel ini, saya hanya akan share bagaimana cara mengetahui orang yang tidak memfollback dan mengunfollow akun tersebut secara grup.

Syarat

Sebelum masuk ke tutorial, pastikan kalian mempunyai laptop atau komputer yang sudah terinstall browser (rekomendasi chrome) dan juga koneksi internet, karena tutorial ini sepenuhnya optimal dijalankan menggunakan komputer.

Langkah-langkah

Di internet ada banyak banget tools dan juga website untuk mengetahui orang yang tidak memfollow akun instagram, tetapi banyak juga yang tools nya dipenuhi dengan iklan dan beberapa juga tidak berfungsi, untungnya saat ini sudah ada developer yang berbaik hati membuatkan script untuk mengatasi masalah ini.

Berikut adalah langkah-langkah nya:

  1. Buka repository github berikut, https://github.com/davidarroyo1234/InstagramUnfollowers
  2. Scroll aja kebawah, pada bagian usage di step 1. Copy the following code, salin code 
  1. Setelah code disalin, buka browser lalu login ke akun instagram kalian instagram.com
  1. Masuk ke console browser, pakai shortcut (ctrl+shift+j)
  1. Paste code yang disalin dari github tadi ke console lalu enter
  1. Setelah itu klik run, selamat mencoba
