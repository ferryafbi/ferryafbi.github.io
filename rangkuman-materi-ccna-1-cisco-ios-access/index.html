Rangkuman Materi CCNA-1: Cisco IOS Access

Ferry Ananda Febian

Dec 15, 2022 • 2 min read

Cisco IOS Access adalah sebuah perangkat lunak yang digunakan pada perangkat Cisco untuk mengatur hak akses pengguna ke jaringan komputer. IOS Access memungkinkan administrator jaringan untuk menentukan siapa saja yang dapat mengakses jaringan, dan bagaimana pengguna dapat mengakses jaringan tersebut.

CLI

Umumnya perangkat Cisco dikonfigurasikan menggunakan antarmuka command line berupa perintah teks untuk mengakses dan mengontrol sistem komputer, hal ini dikarenakan beberapa masalah jaringan tidak dapat diselesaikan melalui GUI atau Graphical User Interface.

Tujuan OS

Sistem operasi adalah perangkat lunak yang bertanggung jawab untuk mengatur dan mengendalikan komputer. Tujuan utama dari sistem operasi adalah untuk memudahkan pengguna dalam mengakses dan mengontrol komputer, serta mengatur sumber daya komputer seperti memori, prosesor, dan perangkat keras lainnya.

Metode Akses

Metode akses adalah cara yang digunakan untuk mengatur dan mengendalikan akses pengguna ke sebuah sistem atau jaringan komputer. 

Berikut beberapa metode akses untuk mengkonfigurasi switch:

  • Console, Console dalam jaringan adalah perangkat atau aplikasi yang digunakan untuk mengelola dan mengontrol perangkat jaringan, seperti router atau switch. Console biasanya terhubung langsung ke perangkat jaringan melalui kabel serial atau koneksi SSH (Secure Shell), dan memungkinkan administrator jaringan untuk mengakses perangkat tersebut dan mengatur pengaturan atau konfigurasi yang diperlukan. Contoh fitur dalam console adalah CLI atau Command Line Interface
  • SSH, atau singkatan dari Secure Shell yaitu sebuah protokol jaringan yang digunakan untuk menghubungkan dua perangkat komputer secara terenkripsi. SSH dalam bidang jaringan digunakan untuk mengakses perangkat jaringan seperti router, switch, atau untuk mengakses server dari komputer jarak jauh.
  • Telnet, Telnet hampir sama seperti SSH hanya saja Telnet dianggap tidak secure atau aman karena koneksi antar perangkat tidak terenkripsi.

Terminal Emulation Programs

Terminal emulation programs adalah perangkat lunak yang digunakan untuk mengakses perangkat jaringan seperti router atau switch melalui koneksi berbasis teks, seperti Telnet atau SSH. Terminal emulation programs biasanya menampilkan antarmuka teks yang mirip dengan command line interface (CLI) pada perangkat jaringan, sehingga pengguna dapat mengetikkan perintah-perintah tertentu untuk mengontrol perangkat jaringan tersebut dari jarak jauh.

Berikut beberapa contoh tools Terminal Emulation:

Putty

Putty adalah tools digunakan untuk mengakses perangkat jaringan seperti router atau switch melalui koneksi SSH (Secure Shell) atau Telnet. Putty memiliki antarmuka yang mudah digunakan dan dapat diinstal pada berbagai sistem operasi, sehingga banyak administrator jaringan yang menggunakan aplikasi ini untuk mengelola jaringan komputer.

Tera Term

Hampir sama seperti Putty, Namun Tera Term hanya support pada sistem operasi Windows dan cocok digunakan untuk pemula.

SecureCRT

Memiliki fungsi yang hampir sama seperti Putty dan Tera Term, hanya saja SecureCRT memiliki banyak fitur lanjutan yang tidak tersedia pada aplikasi terminal emulation lainnya, seperti dukungan untuk protokol jaringan yang lebih banyak, pengaturan konfigurasi yang lebih lengkap, dan fitur-fitur keamanan. SecureCRT cocok digunakan untuk master jaringan.
