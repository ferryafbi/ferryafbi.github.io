Cara Install Jenkins di Linux Redhat dan CentOS

Ferry Ananda Febian

Nov 24, 2022 • 1 min read

Jenkins adalah server otomatisasi open source yang membantu Anda membuat dan menerapkan kode secara otomatis mulai dari pengembangan, pengujian, penerapan integrasi dan pengiriman yang berkelanjutan. 

Berikut adalah langkah-langkah cara install Jenkins di Linux Redhat dan CentOS.

Harap dicatat, untuk proses instalasi Jenkins di Linux Redhat, Anda perlu login ke akun root atau user yang mempunyai akses root.

1. Buat Repository Jenkins

Buka terminal dan jalankan perintah berikut untuk membuat repository Jenkins:

wget -O /etc/yum.repos.d/jenkins.repo http://pkg.jenkins.io/redhat-stable/jenkins.repo
rpm --import http://pkg.jenkins.io/redhat-stable/jenkins.io.key

2. Install Jenkins dan Java

Jalankan perintah berikut untuk menginstall Jenkins dan Java:

yum install jenkins java-1.8.0-openjdk –y

3. Start dan Aktifkan Jenkins

Jalankan perintah berikut untuk untuk start dan aktifkan layanan Jenkins:

systemctl start jenkins; systemctl enable jenkins

4. Buka Port 80 dan 8080 di Firewall Linux

Layanan Jenkins berjalan di port 80 dan 8080. Anda perlu melakukan konfigurasi dengan perintah berikut:

firewall-cmd --zone=public --add-port=8080/tcp --permanent success
firewall-cmd --zone=public --add-service=http --permanent success 
firewall-cmd --reload

5. Test Instalasi Jenkins

Buka browser dan masuk ke http://<ip-server-anda>:8080

http://localhost:8080

Anda akan melihat halaman autentikasi dan diminta untuk memasukkan kata sandi. Kata sandi dapat dilihat di file /var/log/jenkins/jenkins.log. Jalankan perintah berikut untuk melihat kata sandi:

grep -A 5 password /var/log/jenkins/jenkins.log

Salin kata sandi untuk login di halaman autentikasi Jenkins.

6. Konfigurasi Jenkins

Setelah berhasil login, klik Install Suggested Plugins, secara otomatis Jenkins akan menginstall plugins yang direkomendasikan.

Selanjutnya, Anda akan diminta membuat akun Admin. Setelah selesai membuat akun, Klik Start Using Jenkins.
